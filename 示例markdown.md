## 6. 代码块

### 行内代码
使用 `print("Hello World")` 函数

### 代码块（无语法高亮）
```
function example() {
  console.log("Hello World");
}
```

### 带语法高亮的代码块
```javascript
// JavaScript示例
function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}

console.log(fibonacci(10));
```

```python
# Python示例
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(10))
```
